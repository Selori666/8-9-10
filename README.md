Негізгі мүмкіндіктер
	•	Фильмдер тізімін көру
	•	Фильм қосу
	•	Фильм жою
	•	Рейтинг көру
	•	Файлға сақтау (data.txt)
	•	Файлдан оқу
	•	Қате енгізуді өңдеу (try-except)
	•	Рекурсия мысалы (ask_repeat() функциясы)

  Пайдаланылған функциялар мен модульдер

utilis.py
	•	add_film(films, name, rating) — жаңа фильм қосу
	•	delete_film(films, name) — фильмді жою
	•	find_film(films, name) — фильмді табу

sss.py
	•	show_films() — барлық фильмдерді көрсету
	•	save_to_file() — фильмдерді .txt файлға жазу
	•	load_from_file() — файлдан оқу (қате жолдарды өткізіп кетеді)
	•	menu() — негізгі мәзір
	•	ask_repeat() — рекурсия арқылы қайтадан сұрау
